## myVdom.js

innerHTML을 통한 문자열 변경이 아닌 객체를 이용해 가상돔을 출력 후 실제 돔으로 변경하는 작업을 거친다.

만드는 과정에서 황준일님 블로그와 여러 블로그를 참조하며 이해하려 노력하였음

현재 'h'라는 함수를 통해 가상돔을 출력하고 'renderNode'함수를 통해 실제 돔으로 변경하는 작업은 원할하게 이루어진다.
